# Tetris на C++
## Информация
Tetris, который я сделал на C++, используя SFML библиотеку. Он напоминает NES-овскую версию, но музыка прямиком с GameBoy.

Это мой первый проект, который использует SFML, да и в целом первый проект на C++, который я выложил.

## Управление
* **Стрелочки** — передвижение.
* **Z** — переворачивание фигуры.
* **Space** — быстрая установка фигуры.
* **F5** — перезапуск.
* **F4** — (читерский) быстрый переход на новый уровень.

# C++ Tetris
## Information
Tetris game built in C++ using the SFML library. It captures the look and feel of the NES version, while featuring the iconic soundtrack straight from the GameBoy.

This is my very first project utilizing SFML, and my first C++ project ever published.

## Controls
* **Arrow keys** — Move the tetromino.
* **Z** — Rotate the tetromino.
* **Space** — Hard drop (instant placement).
* **F5** — Restart the game.
* **F4** — Next level (cheat code).
